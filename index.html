Piano Practice Daily¶

Requirements¶

Requirement: Detect Piano Start REQ001
links incoming: CMP001

The system shall detect when the electronic piano is turned on.
Requirement: Detect Piano Stop REQ002
links incoming: CMP001

The system shall detect when the electronic piano is turned off.
Requirement: Record Audio REQ003
links incoming: CMP002

The system shall record audio during the practice session.
Requirement: Store Audio on Storage REQ004
links incoming: CMP004

The system shall store the recorded audio either in Google Drive or NAS or Cloud.
Requirement: Remove Silence REQ005
links outgoing: REQ010
links incoming: CMP003

The system shall remove silent periods longer than a specified threshold from the recording. Silent periods are defined as described in id: REQ010.
Requirement: Log Practice Time REQ006

The system shall log the session start time, stop time, and total duration.
Requirement: Send Email Notification REQ007
links incoming: CMP006

The system shall send an email with session information, including start time, stop time, total duration, and a link to the processed audio.
Requirement: Web Portal for History REQ008
links incoming: CMP005

The system shall provide a website that displays daily practice logs and allows playback of past recordings.

  • The web interface shall allow users to view a list of recorded sessions.

  • Users shall be able to play back audio files directly from the web interface.

  • The interface shall display metadata such as session date, duration, and any notes.
Requirement: Real-time Streaming (Optional) REQ009
links incoming: CMP007

The system may support real-time streaming of the piano session.
Requirement: Silence Threshold REQ010
links incoming: REQ005, CMP003

Silence is defined as a continuous period of no audio for more than a configurable number of seconds (e.g., 5 seconds).

Specifications¶

Specification: Piano Detection Module CMP001
links outgoing: REQ001, REQ002

The Piano Detection Module is responsible for detecting when the electronic piano is powered on or off. This module ensures that the system can automatically start and stop recording based on the piano’s state.
Specification: Audio Recording Module CMP002
links outgoing: REQ003

The Audio Recording Module handles the recording of audio during the practice session. It starts recording when the piano is detected as powered on.
Specification: Silence Processing Module CMP003
links outgoing: REQ005, REQ010

The Silence Processing Module processes the recorded audio to remove silent periods longer than a specified threshold.
Specification: Storage Module CMP004
links outgoing: REQ004

The Storage Module is responsible for saving the processed audio files to predefined storage locations such as Google Drive, NAS, or Cloud.
Specification: Web Portal CMP005
links outgoing: REQ008

The Web Portal provides a user interface for reviewing daily practice logs and playing back past recordings. It also displays metadata such as session date, duration, and notes.
Specification: Notification Module CMP006
links outgoing: REQ007

The Notification Module sends email notifications containing session information, including start time, stop time, total duration, and a link to the processed audio.
Specification: Streaming Module (Optional) CMP007
links outgoing: REQ009

The Streaming Module provides real-time audio streaming during the piano session. This module is optional and may not be implemented in all deployments.
